  **1. ES_SER_FORMS** — Set of all conjugated ser forms (accent-stripped variants included for student text robustness):
  `soy, eres, es, somos, sois, son, era, eras, eramos, erais, eran, fui, fuiste, fue, fuimos, fuisteis, fueron, sido`
  Include accent-stripped: `eramos` alongside `éramos` if needed (students often omit accents).

  **2. ES_ESTAR_FORMS** — Set of all conjugated estar forms (accent-stripped variants included):
  `estoy, estas, esta, estamos, estais, estan, estaba, estabas, estabamos, estabais, estaban, estuve, estuviste, estuvo, estuvimos, estuvisteis, estuvieron, estado`
  Include both accented and accent-stripped: `está/esta`, `estás/estas`, `estáis/estais`, `estábamos/estabamos`.

  **3. ES_COPULA_ADJ** — Object mapping adjective base form to required copula (`'ser'`, `'estar'`, or `'both'`):
  - **estar-only** (temporary states, conditions, results): `cansado, enfermo, contento, muerto, sentado, dormido, despierto, preocupado, enamorado, embarazada, harto, enojado, asustado, sorprendido, ocupado, roto, abierto, cerrado, encendido, apagado, mojado, seco, lleno, vacio, limpio, sucio, listo` (when meaning "ready")
  - **ser-only** (inherent qualities): `alto, bajo, grande, pequeno, inteligente, tonto, rico, pobre, joven, viejo, guapo, feo, delgado, gordo, fuerte, debil, importante, necesario, posible, imposible, dificil, facil, obvio, claro, cierto, falso, justo, injusto`
  - **both** (meaning changes): `aburrido, listo, rico, malo, bueno, verde, vivo, seguro, atento, orgulloso`

  Note: Store keys in accent-stripped lowercase form (e.g. `vacio` not `vacío`, `debil` not `débil`) since `ctx.tokens[i].word` is already lowercased and student text often omits accents.

  **4. ES_POR_PARA_TRIGGERS** — Array of trigger-pattern objects for por/para confusion detection. Each entry: `{ id, wrongPrep, correctPrep, context, detect }` where `detect` is a string description of what to look for (the rule file implements the actual detection logic). Include ~12 high-confidence patterns:
  - `por_beneficiary`: `por` + human noun/pronoun → `para` (beneficiary: "por mi familia" → "para mi familia")
  - `por_purpose_inf`: `por` + infinitive (purpose context) → `para` ("por leer" → "para leer")
  - `para_duration`: `para` + duration expression → `por` ("para dos horas" → "por dos horas")
  - `para_cause`: `para` + cause expression → `por` ("para eso llegue tarde" → "por eso llegue tarde")
  - `por_deadline`: `por` + deadline marker → `para` ("por manana" → "para manana")
  - `por_destination`: `por` + destination → `para` ("por Madrid" → "para Madrid" when destination not transit)
  - Additional 6-7 patterns from standard L2 Spanish pedagogy.

  **5. ES_HUMAN_NOUNS** — Set of ~25 common human-denoting nouns (A1-B1 level):
  `madre, padre, hermano, hermana, hijo, hija, amigo, amiga, profesor, profesora, maestro, maestra, doctor, doctora, nino, nina, hombre, mujer, chico, chica, abuelo, abuela, primo, prima, vecino, vecina, companero, companera`
  Store accent-stripped lowercase (e.g. `nino` not `niño`, `companero` not `compañero`).

  **6. ES_COPULA_VERBS** — Set of copula verbs that do NOT take personal "a" (used by ES-03 to skip copula contexts):
  `ser, estar, parecer, resultar, quedarse`

  Add all six new objects/sets to the `tables` export alongside the existing DE tables.

  Update the JSDoc comment at the top to mention ES tables.
