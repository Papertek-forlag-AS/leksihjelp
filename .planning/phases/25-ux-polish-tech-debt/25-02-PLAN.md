<task type="auto">
  <name>Task 2: Subscribe popup to lexi:schema-mismatch and surface Versjonskonflikt banner (DEBT-04)</name>
  <files>extension/popup/popup.js, extension/popup/popup.html</files>
  <action>
Phase 23 wired `vocab-store.js` to broadcast `chrome.runtime.sendMessage({ type: 'lexi:schema-mismatch', cachedSchema, serverSchema, lang })` when the IndexedDB-cached schema_version is older than what the API serves. Today nothing listens for this — the cache silently keeps the baseline and the user has no idea.

Steps:
1. Add a hidden banner element to extension/popup/popup.html, near the top of `#dictionary-view` (or wherever notices belong — match existing notice/banner styling). Suggested markup:
   ```html
   <div id="schema-mismatch-banner" class="lexi-banner lexi-banner-warn hidden" role="alert">
     <span data-i18n="schema_mismatch_title">Versjonskonflikt</span>:
     <span id="schema-mismatch-detail"></span>
   </div>
   ```
2. In extension/popup/popup.js, near other `chrome.runtime.onMessage.addListener` registrations (or, if none, register a fresh listener once at popup load), add:
   ```js
   chrome.runtime.onMessage.addListener((msg) => {
     if (msg && msg.type === 'lexi:schema-mismatch') {
       const banner = document.getElementById('schema-mismatch-banner');
       const detail = document.getElementById('schema-mismatch-detail');
       if (banner) banner.classList.remove('hidden');
       if (detail) detail.textContent =
         `${msg.lang}: cachedSchema=${msg.cachedSchema}, serverSchema=${msg.serverSchema}`;
     }
   });
   ```
3. If the project's i18n strings file has a `schema_mismatch_title` key, reuse it; otherwise hard-code "Versjonskonflikt" as the visible text per REQUIREMENTS.md DEBT-04 wording.
4. Add a minimal CSS class for `.lexi-banner-warn` if no equivalent exists (orange/yellow background, dark text). If `extension/styles/popup.css` already has banner styling, reuse it.

Per CLAUDE.md downstream consumer note: changes to `popup.html`/`popup.js` only affect the extension; no lockdown sync needed for popup-only files.
  </action>
  <verify>
    <automated>grep -q "lexi:schema-mismatch" extension/popup/popup.js && grep -q "schema-mismatch-banner" extension/popup/popup.html</automated>
  </verify>
