**Rule file (fr-subjonctif.js):**

Create following the IIFE pattern from fr-etre-avoir.js / fr-elision.js. Key properties:
- `id: 'fr-subjonctif'`
- `languages: ['fr']`
- `priority: 60` (same tier as ES subjuntivo)
- `severity: 'warning'`

**Detection algorithm:**
1. Iterate sentences via `ctx.sentences`. For each sentence, get tokens via `tokensInSentence(ctx, si)`.
2. Walk tokens looking for multi-word trigger matches against `FR_SUBJONCTIF_TRIGGERS`. Same approach as ES: at each token position, concatenate 2-5 consecutive tokens (lowercased, space-joined) and check if any prefix matches a trigger in the Set. All triggers end with "que".
3. After matching a trigger ending in "que", scan forward past optional subject pronouns (je, tu, il, elle, on, nous, vous, ils, elles) and articles/possessives.
4. Find the next token that is a known verb form via `ctx.indexes.frPresensToVerb`. If the token (lowercased) is in this Map, it's a presens/indicative form.
5. Look up `{ inf, person }`. Then check `ctx.indexes.frSubjonctifDiffers.get(inf + '|' + person)`:
   - If `false` or `undefined`: do NOT flag. The forms are homophonous (e.g., "parle" for parler je). The student may have written the correct subjunctive form -- we cannot tell.
   - If `true`: the forms differ. Look up `ctx.indexes.frSubjonctifForms.get(inf + '|' + person)` to get the subjunctive form. Flag with suggestion.

**CRITICAL: The homophony guard is the defining feature of this rule.** Without it, every regular -er verb after "il faut que je/tu/il/ils" would be a false positive. The research confirms: regular -er je/tu/il/ils forms are identical in presens and subjonctif. Only nous/vous differ for regular verbs; all persons differ for irregular verbs.

**Examples:**
- "Il faut que je parle mieux" -> `frPresensToVerb.get('parle')` = `{inf:'parler', person:'je'}` -> `frSubjonctifDiffers.get('parler|je')` = `false` -> NO flag (correct)
- "Il faut que je fais mes devoirs" -> `frPresensToVerb.get('fais')` = `{inf:'faire', person:'je'}` -> `frSubjonctifDiffers.get('faire|je')` = `true` -> `frSubjonctifForms.get('faire|je')` = `'fasse'` -> FLAG: "fais" -> "fasse"
- "Avant que nous parlons" -> `frPresensToVerb.get('parlons')` = `{inf:'parler', person:'nous'}` -> `frSubjonctifDiffers.get('parler|nous')` = `true` -> FLAG: "parlons" -> "parlions"

**explain function:** Return `{ nb, nn, severity: 'warning' }`. Explain that after the trigger phrase, subjunctive mood is required. Use `escapeHtml` on interpolated values.
