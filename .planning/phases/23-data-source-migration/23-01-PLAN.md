GET /api/vocab/v1/bundle/{language} → 200 application/json
{
  "schema_version": 1,
  "revision": "2026-04-27T08-00-00Z-de-r3",  // monotonic per-language; date+slug ok

  "freq": {...}, "bigrams": {...},
  "grammarFeatures": [...],
  "falseFriends": [...],
  "senses": {...}
}
404 if language unknown. 500 on internal failure.

GET /api/vocab/v1/revisions → 200 application/json
{
  "schema_version": 1,
  "revisions": { "nb": "...", "nn": "...", "de": "...", "es": "...", "fr": "...", "en": "..." }
}

CORS (both endpoints, all responses including OPTIONS preflight):
  Access-Control-Allow-Origin: <echo of request Origin if it matches chrome-extension://* or https://leksihjelp.no, else omit>
  Access-Control-Allow-Methods: GET, OPTIONS
  Access-Control-Allow-Headers: Content-Type, If-None-Match
  Access-Control-Max-Age: 86400
Preflight (OPTIONS): respond 204 with the headers above and no body.
