<objective>
Build the bootstrap path so a fresh install works:
1. A trimmed `nb-baseline.json` (≤ 200 KB) replaces the full `nb.json` as the bundled fallback.
2. A new `background/vocab-bootstrap.js` (loaded by the service worker) auto-downloads the user's selected target language(s) into IndexedDB on install/upgrade.
3. The popup surfaces download progress/success/failure state per language, falling back gracefully to the NB baseline on failure.

Purpose: Without this, removing bundled `de/es/fr/en` data (plan 05) breaks first-run UX.
Output: Baseline build script + baseline JSON + bootstrap service module + popup status UI.
</objective>
