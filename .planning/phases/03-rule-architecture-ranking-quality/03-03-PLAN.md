Step 1 — Edit `extension/content/spell-rules/nb-typo-fuzzy.js` (file created in Plan 02).

Replace the `scoreCandidate` reference (whether it's imported from `__lexiSpellCore` or a local copy, depending on how Plan 02 structured it) with a local Zipf-aware version declared INSIDE the IIFE. Reasoning: the fuzzy rule is the only consumer of scoring; keeping it local enforces single-responsibility and simplifies future tuning without touching core.

```javascript
// Scoring heuristic — higher is better. Adds a bounded Zipf tiebreaker to the
// base formula from core: higher-frequency NB/NN words win when everything
// else ties. SC-01 (Phase 3-03).
//
// Pitfall 4 (RESEARCH.md:420-432): multiplier MUST be bounded so a d=2
// common-word never beats a d=1 rare-word. Max Zipf ≈ 7 → max boost 105.
// Distance penalty per edit is 100. So: d=1 Zipf-0 scores -100; d=2 Zipf-7
// scores -200 + 105 = -95. d=1 still wins comfortably.
function scoreCandidate(query, cand, d, vocab) {
  const pref = sharedPrefixLen(query, cand);
  const suff = sharedSuffixLen(query, cand);
  let s = pref * 15 + suff * 10 - d * 100;
  if (cand.length < query.length) s -= 50;
  if (isAdjacentTransposition(query, cand)) s += 40;

  // Zipf tiebreaker — vocab.freq is populated for NB/NN by Phase 3-01
  // (freq-{lang}.json sidecar). Empty Map for other languages — which is fine
  // because fuzzy is NB/NN-only anyway.
  if (vocab && vocab.freq) {
    const z = vocab.freq.get(cand);
    if (typeof z === 'number') s += z * 15;
  }
  return s;
}
```

Step 2 — Update the fuzzy neighbor lookup to pass `vocab` through. The existing `findFuzzyNeighbor(word, validWords)` takes only two args; change to `findFuzzyNeighbor(word, vocab)` and pull `validWords = vocab.validWords` inside:

```javascript
function findFuzzyNeighbor(word, vocab) {
  const validWords = vocab.validWords;
  const len = word.length;
  const k = len <= 6 ? 1 : 2;
  let best = null, bestScore = -Infinity;
  const first = word[0];
  for (const cand of validWords) {
    const cl = cand.length;
    if (Math.abs(cl - len) > k) continue;
    if (cand[0] !== first) continue;
    if (cand === word) continue;
    const d = editDistance(word, cand, k);
    if (d > k) continue;
    const score = scoreCandidate(word, cand, d, vocab);
    if (score > bestScore) { bestScore = score; best = cand; }
  }
  return best;
}
```
