<objective>
Build the in-popover Lær mer UI: a secondary button that expands a teaching panel rendering the pedagogy block from the finding. This is the visible deliverable of phase 26 — the data and rule hookup land in 26-01, the structural gate lands in 26-02, and 26-03 makes it appear in front of the student.

Purpose: Students who hit a DE preposition correction get *why*, not just *what*. The expansion stays inside the existing popover surface (no modal, no side-panel, no overlay) so dragging behavior and Tab navigation keep working.

Output:
- New i18n keys covering all student-facing strings inside the panel.
- spell-check.js popover render extended to emit the Lær mer button + collapsed panel scaffold; a toggle handler that builds and inserts panel content lazily on first click.
- content.css with case-colour-coded badges, correct/incorrect rows with both colour and icon, motion/location pair styled distinctly, colloquial aside as a friendly light-background italic aside.
- Version bump (manifest + package.json + landing page) per release workflow.
