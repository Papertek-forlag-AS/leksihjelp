## Blocking Issues

### 1. Phase 15 Not Executed (COLL-01 through COLL-04)

Phase 15 (Collocations at Scale) is listed in ROADMAP.md and REQUIREMENTS.md but has no phase directory, no plans, and no execution. Four requirements are orphaned:

- **COLL-01**: NB preposition collocations (`flink i → til`, `glad på → i`)
- **COLL-02**: DE preposition collocations
- **COLL-03**: FR preposition collocations
- **COLL-04**: ES preposition collocations

### 2. Vocab-Seam Browser Wiring Gap (9 Rules Silent)

`spell-check.js` builds a manual `vocab` object from individual `VOCAB.get*()` calls but is missing 8 indexes added by Phases 8-14. Rules guard with `if (!index) return []` — passing fixtures perfectly (which use `buildIndexes()` directly) but producing zero findings in the browser.

**Missing indexes:**
- `participleToAux` — DE-03, FR-02, FR-03
- `esPresensToVerb`, `esSubjuntivoForms`, `esImperfectoForms`, `esPreteritumToVerb` — MOOD-01, MOOD-02, PRON-01, PRON-02
- `frPresensToVerb`, `frSubjonctifForms`, `frSubjonctifDiffers` — MOOD-03
- `irregularForms` — MORPH-01

**Fix:** Add 8 new getters to `vocab-seam.js` + wire into `spell-check.js` vocab object (lines 225-246).

### 3. doc-drift-de-address.js Deleted From Working Tree

File committed in `a14f064` but deleted (unstaged) from the working tree. Causes 3 release gates to fail (`check-explain-contract`, `check-rule-css-wiring`, `check-benchmark-coverage`).

**Fix:** `git checkout HEAD -- extension/content/spell-rules/doc-drift-de-address.js`
