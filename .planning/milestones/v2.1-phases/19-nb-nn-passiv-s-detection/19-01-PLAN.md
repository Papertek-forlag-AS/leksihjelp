Create `/Users/geirforbord/Papertek/papertek-vocabulary/scripts/enrich-nb-nn-s-passive.js` that:

**NB s-passive generation (~671 verbs):**
- For each verb in NB verbbank with conjugations.presens.former.infinitiv:
  - Extract bare infinitive (strip leading "a " or "aa "). Handle array values (take first).
  - Generate s_passiv_infinitiv: bare_inf + "s" (e.g., "skrive" -> "skrives", "lese" -> "leses")
  - Generate s_passiv_presens: same as s_passiv_infinitiv (NB s-passive presens = infinitive + s)
  - Skip if bare_inf length <= 2 (avoid "se" -> "ses" etc. per Pitfall 7)
  - Skip entries that already have isDeponent: true
  - Write forms into conjugations.presens.former as string values

**NN s-passive generation (~622 verbs):**
- For each verb in NN verbbank with conjugations.presens.former.infinitiv:
  - Extract bare infinitive (strip "aa "). Handle array values (take first, or generate for each).
  - Generate s_passiv_infinitiv: if bare_inf ends in "a" -> bare_inf + "st" (e.g., "skriva" -> "skrivast"); if bare_inf ends in "e" -> replace "e" with "ast" (e.g., "skrive" -> "skrivast")
  - For verbs with dual infinitives (array like ["aa akseptera", "aa akseptere"]), generate s-passive for each form
  - Skip if bare_inf length <= 2
  - Skip entries that already have isDeponent: true

**Deponent marking (NB):**
- Load ordbank cache from `vocabulary/ordbank/cache/nb-lookup.json`
- Extract all verb entries whose lemma (key before |verb) ends in "s" or "es" — these are inherent -s verbs
- For each matching entry in NB verbbank (match by word or by word + "s"), set `isDeponent: true`
- Also hardcode a fallback list of high-priority NB deponents: finnes, synes, lykkes, trives, ferdes, minnes, skjelves (in case ordbank matching misses some)

**Deponent marking (NN):**
- Load ordbank cache from `vocabulary/ordbank/cache/nn-lookup.json`
- Extract all verb entries with paradigme_id "429" (the st-verb paradigm, confirmed by the 279 -ast entries)
- For each matching entry in NN verbbank (match by word), set `isDeponent: true`
- Also hardcode high-priority NN st-verbs: synast, finnast, trivast, lykkast, minnast, kjennast, moetast, slast

**Script structure:** Use Node.js with fs.readFileSync/writeFileSync (matching existing enrichment scripts like enrich-nb-complete.js). Print summary: "NB: X s-passive forms added, Y deponents marked. NN: X s-passive forms added, Y deponents marked."

Run the script after creation: `cd /Users/geirforbord/Papertek/papertek-vocabulary && node scripts/enrich-nb-nn-s-passive.js`

Verify output by spot-checking: grep for "s_passiv_infinitiv" in both verbbank files, count occurrences, confirm "isDeponent" appears.
