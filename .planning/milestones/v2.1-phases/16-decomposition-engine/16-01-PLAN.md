**decomposeCompound function** (~80 LOC, place before buildIndexes):

Signature: `function decomposeCompound(word, nounGenus, lang, depth = 0)`

Algorithm (from 16-RESEARCH.md, both-sides greedy recursive):
1. Guard: if depth > 3, return null (max 4 components = max 3 splits)
2. Guard: if word.length < 6, return null (min 3+3)
3. Guard: if nounGenus.has(word), return null (stored entry takes precedence)
4. Get linkers = LINKERS_BY_LANG[lang] || [] 
5. Scan split positions from 3 to word.length - 3:
   a. left = word.slice(0, splitPos)
   b. If NOT nounGenus.has(left): continue
   c. remainder = word.slice(splitPos)
   d. **Zero-fuge direct match:** if nounGenus.has(remainder), return 2-part result
   e. **Zero-fuge recursive:** sub = decomposeCompound(remainder, nounGenus, lang, depth+1); if sub, prepend left and return
   f. **For each linker:** if remainder starts with linker, strip it -> stripped. If stripped.length < 3 continue. If nounGenus.has(stripped), return 2-part result with linker. Else recursive on stripped.
   g. **Triple-consonant elision:** if left ends with char c AND (c + remainder) yields a known noun in nounGenus OR a valid recursive decomposition, return with elision noted. Specifically: if left[left.length-1] === left[left.length-2] (last two chars same), let restored = left[left.length-1] + remainder. If nounGenus.has(restored), return result. If decomposeCompound(restored, ...) returns non-null, use that.
6. Return null

**Return shape:**
```javascript
{
  parts: [
    { word: 'hverdag', genus: 'm', linker: 's' },
    { word: 'mas', genus: 'm', linker: '' }
  ],
  gender: 'm',       // genus of last part
  confidence: 'high'
}
```
The last part always has `linker: ''`. Intermediate parts carry their trailing linker.
