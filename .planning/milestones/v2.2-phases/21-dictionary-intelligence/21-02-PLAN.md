**sanitizeWarning helper:**
Add inside the IIFE, near other helper functions:
```javascript
function sanitizeWarning(html) {
  return html
    .replace(/<(?!\/?(?:em|strong)\b)[^>]*>/gi, '')
    .replace(/&lt;(\/?)(em|strong)&gt;/gi, '<$1$2>');
}
```
