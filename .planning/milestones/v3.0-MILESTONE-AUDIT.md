## Integration Findings

### SCHEMA-01 Developer-View Gap (non-blocking)

`vocab-store.js` emits `lexi:schema-mismatch` via `chrome.runtime.sendMessage` but `popup.js` has no subscriber. On schema mismatch, user sees generic "Ordlister utilgjengelig — prøv igjen senere" instead of "Versjonskonflikt" diagnostic. Cache preservation works correctly. Gap is dormant while schema_version=1 on both client and server.

### Stale BUNDLED_LANGS (cosmetic)

`vocab-seam.js` lists `['nb', 'nn', 'en']` as bundled languages, but `nn.json` and `en.json` were deleted in plan 23-05. No runtime break — the IDB path handles nn/en correctly. Only affects the rare no-store fallback path.

### check-fixtures Gate Status

23-VERIFICATION.md states "check-fixtures exits 0" but the gate currently exits 1 due to pre-existing failures in 5 suites outside Phase 23 scope (de/doc-drift, nb/homophone, nb/saerskriving, nn/typo, de/verb-final). These are carry-over defects, not v3.0 regressions.
