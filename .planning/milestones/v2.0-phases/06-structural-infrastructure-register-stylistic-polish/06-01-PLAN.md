  **A. spell-check-core.js — extend check() function:**

  1. After `const tokens = tokenize(text);` and before `const ctx = ...`, add sentence segmentation:
     ```javascript
     let sentences = [];
     if (typeof Intl !== 'undefined' && Intl.Segmenter) {
       const segmenter = new Intl.Segmenter(lang, { granularity: 'sentence' });
       sentences = [...segmenter.segment(text)].map(seg => ({
         text: seg.segment,
         start: seg.index,
         end: seg.index + seg.segment.length,
       }));
     } else {
       sentences = [{ text, start: 0, end: text.length }];
     }
     ```

  2. Extend the ctx object to include sentences and suppressedFor:
     ```javascript
     const ctx = {
       text, tokens, vocab: vocabRef, cursorPos, lang,
       suppressed: new Set(),
       suppressedFor: { structural: new Set() },
       sentences,
     };
     ```

  3. In the findings loop where `rule.check(ctx)` output is pushed, also stamp `severity` on each finding from the rule:
     ```javascript
     const out = rule.check(ctx);
     if (Array.isArray(out) && out.length) {
       for (const f of out) {
         if (!f.severity && rule.severity) f.severity = rule.severity;
       }
       findings.push(...out);
     }
     ```
