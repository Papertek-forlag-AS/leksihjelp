3. In `buildIndexes()` (line ~824), after the existing `buildLookupIndexes` call and `buildParticipleToAux`, add a new function `buildMoodIndexes(raw, lang)` that builds six reverse-lookup Maps from the RAW verbbank data (NOT from wordList -- must use unfiltered `raw` parameter directly, same pattern as `buildParticipleToAux`):

   For ES (lang === 'es'):
   - `esPresensToVerb`: Map of presens conjugated form (lowercase + accent-stripped variant) -> `{ inf: infinitive, person: pronounKey }`. Built by iterating `raw.verbbank`, reading each verb's `conjugations.presente` object. For each pronoun key (yo, tu, el/ella/usted, etc.) and its form, add both the accented form and `form.normalize('NFD').replace(/[̀-ͯ]/g, '')` (accent-stripped) to the map.
   - `esSubjuntivoForms`: Map of `infinitive|personKey` -> subjuntivo form. Built by iterating `raw.verbbank`, reading `conjugations.subjuntivo`. Accent-strip the lookup key's person portion is NOT needed -- the key uses the canonical pronoun.
   - `esImperfectoForms`: Map of `infinitive|personKey` -> imperfecto form. Same pattern from `conjugations.imperfecto`.
   - `esPreteritumToVerb`: Map of preterito conjugated form (lowercase + accent-stripped) -> `{ inf: infinitive, person: pronounKey }`. Built from `conjugations.preterito`.

   For FR (lang === 'fr'):
   - `frPresensToVerb`: Map of presens form (lowercase) -> `{ inf: infinitive, person: pronounKey }`. Built from `conjugations.presens`.
   - `frSubjonctifForms`: Map of `infinitive|personKey` -> subjonctif form. Built from `conjugations.subjonctif`.
   - `frSubjonctifDiffers`: Map of `infinitive|personKey` -> boolean. For each verb+person, set `true` if `subjonctifForm !== presensForm` (both lowercased). This is the homophony guard -- Plan 03's FR rule will only flag when this is `true`.

4. Include ALL six (or seven with frSubjonctifDiffers) indexes in the returned object from `buildIndexes()`, alongside existing indexes like `participleToAux`, `knownPresens`, etc. For non-matching languages, return empty Maps (e.g., `esPresensToVerb` is empty Map when lang !== 'es').

5. The pronoun keys in ES data use: `yo`, `tú`, `él/ella/usted`, `nosotros`, `vosotros`, `ellos/ellas/ustedes`. The pronoun keys in FR data use: `je`, `tu`, `il/elle`, `nous`, `vous`, `ils/elles`. Use these EXACTLY as they appear in the JSON conjugation objects.

IMPORTANT: Build these indexes from `raw` (the unfiltered vocab data parameter), NOT from `wordList` or `unfilteredWordList`. This follows the `buildParticipleToAux(raw)` pattern and avoids the Phase 05.1 bug where feature-gated wordList starved lookup indexes.
