## Cross-Phase Integration

**Wiring integrity:** All 18 cross-phase integration touchpoints identified by the integration checker are wired correctly. No orphaned exports, no missing connections, no broken flows.

**Pipeline trace:** `vocab-seam-core.js` → `vocab-seam.js` (self.__lexiVocab) → `spell-check.js` (runCheck vocab literal) + `word-prediction.js` → 7 spell-rules plugins (registered via `self.__lexiSpellRules`) → `spell-check-core.js` runner → DOM popover via `renderExplain` 3-way lookup. Manifest content-script order enforces load-time invariants. Live toggle `spellCheckAlternatesVisible` round-trips through `chrome.storage.local` + `chrome.storage.onChanged` for popover re-render.

**SC-01 integration gap closed:** The 2026-04-20 audit identified a defect where `spell-check.js:runCheck()` assembled a 5-field vocab object that omitted `freq`, causing the Zipf tiebreaker in `nb-typo-fuzzy.js` to silently no-op in the browser while passing in the fixture runner. Phase 03.1 closed this with `VOCAB.getFreq()` in the vocab literal + a source-regex adapter-contract guard in `check-fixtures.js` that now fires loud on regression.

**E2E flows verified complete (5/5):**

1. **Type a typo in NB → Zipf-ranked fuzzy suggestion → student-friendly explain copy** — `hagde` → `hadde` top-ranked via `vocab.freq.get(cand) * ZIPF_MULT`; popover renders "`hagde` står ikke i ordboken — kanskje du mente `hadde`". Chrome smoke-test confirmed 2026-04-20 + 2026-04-21.
2. **Type prefix → top-3 word-prediction with Vis-flere reveal** — `VISIBLE_DEFAULT=3` + `VISIBLE_EXPANDED=8`; Vis-flere click + ArrowDown auto-reveal both work. Chrome smoke-test PASS.
3. **NN word typed in NB document → nb-dialect-mix flags with suggestion** — `ikkje` → sky-blue underline + popover "`ikkje` er nynorsk — prøv `ikke`" with Bokmål register badge. Chrome smoke-test PASS.
4. **English paragraph pasted in NB → at most 1 flag per paragraph** — `nb-codeswitch.js` (priority 1, density window) populates `ctx.suppressed`; downstream rules honor suppression. Fixture 13/13 + 500-word clean article 558 words 19/19.
5. **Toggle "Vis alternative skriveforslag" → popover live re-renders multi-suggest** — `chrome.storage.onChanged` subscriber swaps single-suggestion branch for top-3 button rows with Vis-flere reveal. Chrome smoke-test PASS.
