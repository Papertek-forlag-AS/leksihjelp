Perfekt for elever med dysleksi

  • Stavekontroll med tydelige forklaringer — ikke bare røde streker
  • Smarte ordforslag som forstår skrivefeil
  • Hør uttalen for å bekrefte riktig ord
  • Kontekstbaserte forslag (f.eks. riktig verbform etter pronomen)
  • Tilpassbare grammatikkinnstillinger for å redusere informasjonsmengde
  • Fungerer offline — ingen nettforbindelse nødvendig for ordbok

Gratis og åpen kildekode

Ordbok, stavekontroll, ordforslag og grammatikk fungerer offline uten konto. For uttale kan du velge mellom gratis nettleser-stemmer eller naturlige AI-stemmer via abonnement.

Manuell installasjon

Foretrekker du å installere manuelt? Last ned ZIP-filen og følg stegene under.

Last ned ZIP
▶ Installasjonssteg
  1. Last ned og pakk ut ZIP-filen
  2. Åpne Chrome og gå til chrome://extensions
  3. Slå på Utviklermodus (øverst til høyre)
  4. Klikk Last inn upakket
  5. Velg mappen du pakket ut

Nytt i versjon 2.5

Stavekontroll med 54+ regler for alle seks språk, toveis oppslag mellom alle språkpar inkludert bokmål↔nynorsk, sammensatt-ord-intelligens for tyske og norske sammensetninger, sidepanel for å holde ordboken synlig mens du skriver, og mye mer.
