# Personvernregler for Leksihjelp

*Sist oppdatert: 7. februar 2026*

## Hva er Leksihjelp?

Leksihjelp er en nettleserutvidelse som hjelper norske elever med å slå opp ord, høre uttale og få ordforslag når de lærer spansk, fransk eller tysk.

## Hvilke data samler vi inn?

### Data som behandles midlertidig
- **Markert tekst**: Når du markerer tekst på en nettside og bruker Leksihjelp, sendes teksten til vår server for å hente uttale (tekst-til-tale). Denne teksten lagres ikke permanent.

### Data som lagres lokalt
- **Brukerinnstillinger**: Valgt språk og avspillingshastighet lagres lokalt i nettleseren din (chrome.storage). Disse dataene forlater aldri enheten din.

## Hva vi IKKE gjør

- Vi selger ikke dine data til tredjeparter
- Vi samler ikke inn personlig identifiserbar informasjon
- Vi sporer ikke din nettaktivitet
- Vi bruker ikke informasjonskapsler (cookies)
- Vi lagrer ikke teksten du slår opp

## Tredjepartstjenester

Vi bruker ElevenLabs for tekst-til-tale-funksjonalitet. Teksten du markerer sendes til deres API for å generere uttale. Se [ElevenLabs' personvernregler](https://elevenlabs.io/privacy) for mer informasjon.

## Datalagring

Alle brukerinnstillinger lagres lokalt på din enhet. Vi har ingen brukerkontoer eller servere som lagrer personlige data.

## Dine rettigheter

Du kan når som helst:
- Slette utvidelsen og alle lokalt lagrede data
- Se hvilke innstillinger som er lagret via nettleserens utviklerverktøy
